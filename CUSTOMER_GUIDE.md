## Troubleshooting

### App won't start
- Delete the executable
- Re-download and try again
- Make sure Windows Defender isn't blocking it

### Data disappeared
- Check if you accidentally reset (Ctrl+Shift+R)
- Cloud sync? Check if you're logged in
- Look in: `C:\Users\YourName\AppData\Roaming\deskday`

### Slow startup
- First run takes 10-15 seconds (normal)
- Subsequent launches are instant
- Make sure no other heavy apps are running

### Can't edit schedule
- Click the hour cell to open editor
- Max 3 lines per hour
- Close with button or ESC
